## Local web server

Here you can find all the files and directories to run the demo in a local web server like Apache2.

Please, note that you need:

1. Python 3.7 or above
2. The following list of libraries installed:
	* numpy
	* pandas
	* bs4
	* tensorflow
	* sklearn
